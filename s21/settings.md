# Настройка Homebrew

Коротко:

```bash
cd ~/goinfre
git clone https://github.com/Homebrew/brew homebrew
eval "$(homebrew/bin/brew shellenv)"
brew update --force --quiet
chmod -R go-w "$(brew --prefix)/share/zsh"
cd
brew install lcov
```
